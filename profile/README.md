# Puslegutta

Interactive quiz and puzzle games — built to teach, challenge, and entertain.

## Apps

| App | Description |
|-----|-------------|
| [Pinpoint](https://github.com/Puslegutta/Pinpoint) | Interactive quiz platform with 3D globe, anatomy, solar system, and more |
| [Iddiotquiz](https://github.com/Puslegutta/Iddiotquiz) | Norwegian quiz game with transition word chains and a brain economy |
| [Wrd](https://github.com/Puslegutta/Wrd) | Word puzzle game |

| Repo | Description |
|------|-------------|
| [PusleguttaCMS](https://github.com/Puslegutta/PusleguttaCMS) | Sanity CMS for quiz content shared across apps |
